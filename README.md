# Root-Colonization-Counter
An HTML Widget created by iterative, user-driven development supported by AI-assisted prototyping.

# Use Case

The application is designed to integrate seamlessly into the grid intersect microscopy process. 

Before scoring, the microscopist should evaluate a few slides to determine the proper initial categories (this can be changed later if new species or structures are seen). 

Then, buttons can be named with the categories and the user can tie keyboard keys to each button using the ‘Key Presets’ menu.

During scoring, the microscopist visualizes root segments and classifies structures using the keyboard. Audio feedback will confirm each entry, allowing uninterrupted visual focus once key locations are memorized. 

When the user arrives at their chosen amount of intersections, (this can be tracked by the ‘Total:’ at the top of the page) press ‘save score’ and the sample can be named and notes can be entered. 

Exported datasets are structured in a wide (pivoted) format.
